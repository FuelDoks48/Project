Описание курса

В первой части курса вы познакомитесь со стандартами Web 2.0,научитесь работать с макетами и форматировать код,освоите адаптивную вёрстку,а по итогу — сможете создавать быстрые и удобные сайты,которые точно понравятся пользователям.

Вы поймете назначения основных тегов HTML,сможете без усилий стилизоватьи размещать элементы с помощью CSS и сверстаете свой первый макет.

Вторая часть курса сделает из вас настоящего мастера. Вы изучите сборщики проектов,научитесь верстать email-рассылки,разберетесь с анимацией на сайте.

Помимо этого,вы сверстаете фрагмент из интернет-магазина — главную страницу,старинцу каталога и страницу товара!
